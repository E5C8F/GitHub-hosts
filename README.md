# GitHubSpeedHosts


Windows上，基于hosts的GitHub加速项目，自动检测GitHub并写入最快IP到host以解决部分地区的GitHub访问问题。


使用方法：


下载github加速（hosts）.bat文件并执行。

[下载](https://github.com/E5C8F/GitHubSpeedHosts/blob/main/github加速（hosts）.bat)
